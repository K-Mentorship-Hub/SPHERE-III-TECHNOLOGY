# Practice Tasks · SPHERE III / TECHNOLOGY

## Starter Tasks

### `T1` · AI / Data / Analytics

1. Build a tiny dataset and make one visual insight from it.
2. Compare three tools for the same task and explain when to use each one.
3. Create a short `AI productivity starter stack` guide.

### `T2` · Software / Engineering / Automation

1. Turn a repetitive process into a simple workflow diagram.
2. Build a tiny automation idea as a checklist or pseudo-flow.
3. Write a `first repo setup` guide for beginners.

### `T3` · Dashboards / Interfaces / Open Tools

1. Make a static dashboard mockup for one chosen direction.
2. Improve the readability of one public page or README.
3. Draft a user-facing explanation for one technical tool.

### `T4` · Infra / Reproducibility / Technical Ops

1. Create a reproducibility checklist for a small project.
2. Compare `manual workflow` vs `structured workflow`.
3. Draft a `how to keep your build process clean` note.
